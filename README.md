2:1 Heatshrinks should be okay. Even for USB A

Connector | rec. Size
--------- | ---------
USB A     | 9.5mm
Micro USB | 6.4mm
Mini USB  | 6.4mm

### Paracord

2mm is fine for the above mentioned cable
